# Expanded Labs: Practical Cybersecurity Skill Building

These expanded labs demonstrate practical exposure to SOC operations, endpoint security, cloud IAM, vulnerability management, network defense, incident response, and security documentation. Each project is simulated, ethical, and home-lab based.

These labs support the main featured projects and show continuous hands-on learning across:

- SOC operations
- endpoint security
- network defense
- cloud IAM
- vulnerability management
- Active Directory
- packet analysis
- incident response
- detection engineering
- governance and compliance
