# Cybersecurity Portfolio

This repository showcases thirty realistic, beginner-friendly cybersecurity projects designed for entry-level SOC Analyst and Cybersecurity Specialist roles. All scenarios are simulated in a safe home-lab environment.

## Core Portfolio Projects

1. SOC Analyst Home Lab
2. Active Directory Security Lab
3. Vulnerability Management Lab
4. Network Monitoring and Packet Analysis Lab
5. Cloud Security IAM Lab

## Expanded Project Set

An additional set of 25 hands-on projects is included to reflect practical enterprise-style SOC and IT security work across detection engineering, incident response, cloud, endpoint, network defense, compliance, and automation.

See the full list in additional-projects-index.md.

## Skills Demonstrated

- Security monitoring and alert triage
- Incident response workflow and documentation
- Log analysis across Windows and Linux systems
- Vulnerability scanning, prioritization, and remediation planning
- Network traffic analysis and threat hunting
- IAM hardening and cloud misconfiguration detection

## Important Safety Note

All attack activity, suspicious behavior, and logs in this repository are simulated and intended only for controlled lab environments. No instructions are provided for unauthorized or harmful use.

## Recommended Use

- Add screenshots from your own lab runs in each project screenshot folder.
- Use the provided report templates to produce portfolio artifacts for interviews.
