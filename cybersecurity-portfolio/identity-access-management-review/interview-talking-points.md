# Interview Talking Points

## Explain the Project in 30 Seconds

This project simulates real SOC and security operations tasks using safe lab telemetry and repeatable analyst workflows. I focused on detection quality, evidence correlation, and practical remediation.

## What I Learned

- How to turn noisy telemetry into prioritized incidents.
- How to map technical events to ATT and CK techniques.
- How to communicate findings clearly to both technical and non-technical stakeholders.

## How This Applies to a Real SOC Role

- Faster triage and cleaner handoffs.
- Better incident documentation and root-cause analysis.
- Continuous improvement in detection engineering and security controls.
