# Resume Bullets for Cybersecurity Portfolio

## SOC Analyst Home Lab

- Built a simulated SOC workflow using SIEM alerts, Windows logs, and Linux authentication logs to triage suspicious activity and document containment actions.
- Investigated brute-force and privilege escalation indicators, mapped findings to MITRE ATT&CK techniques, and produced an incident summary with response recommendations.

## Active Directory Security Lab

- Audited Active Directory security events (failed logons, account lockouts, privilege group changes) to identify risky identity activity and hardening opportunities.
- Developed a repeatable AD investigation checklist using Event IDs and PowerShell queries to improve account monitoring and incident response readiness.

## Vulnerability Management Lab

- Performed vulnerability scanning with Nessus/OpenVAS in a controlled environment, prioritized findings by CVSS and business risk, and drafted remediation timelines.
- Created a vulnerability report with asset criticality, patch validation steps, and risk-accepted exceptions for stakeholder review.

## Network Monitoring and Packet Analysis Lab

- Used Wireshark and Nmap results to analyze suspicious traffic patterns, detect unauthorized scanning behavior, and validate segmentation controls.
- Documented packet-level indicators, protocol anomalies, and recommended monitoring rules to improve network threat visibility.

## Cloud Security IAM Lab

- Reviewed IAM policies for least-privilege alignment, identified over-permissive roles, and proposed corrective controls for production-like cloud accounts.
- Produced a cloud IAM misconfiguration report including remediation actions, MFA enforcement strategy, and continuous auditing recommendations.
