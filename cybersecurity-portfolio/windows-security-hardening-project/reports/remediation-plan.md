# Remediation Plan

## Immediate Actions

- Contain affected host or identity.
- Block confirmed malicious indicators.
- Reset or rotate potentially exposed credentials.

## Medium-Term Improvements

- Tune noisy detections and add enrichment context.
- Improve logging fidelity and retention.
- Update playbooks and escalation paths.

## Validation

- Re-run simulation to confirm controls detect and contain behavior.
- Record false-positive and true-positive rate changes.
